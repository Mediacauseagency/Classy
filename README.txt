A tool used to integrate Classy's (https://classy.org) API into easily accessible shortcodes

== Description ==

Classy is the world’s largest fundraising platform for social good organizations. Since 2011, we’ve enabled 1.3 million people across 300K individual campaigns to raise over $130M for social good organizations. From cutting-edge health programs to educational advancement, our customers are tackling the world’s greatest challenges with the power of the Classy platform.

This plugin integrates the Classy API via shortcodes for easy access on Wordpress.  

Currently supported shortcodes/API calls.

[classy_campaigns]
[classy_fundraisers]
[classy_donations]

== Installation ==

Upload the Classy plugin to your blog, activate it, then enter your API Token and CID in the Classy menu.
